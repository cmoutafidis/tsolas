# Αποτελέσματα 1ου Ηλεκτρονικού Διαγωνισμού Διηγήματος
Σήμερα Παρασκευή, 14 Φεβρουαρίου 2014, ανακοινώνονται τα αποτελέσματα του 1ου Ηλεκτρονικού Διαγωνισμού Διηγήματος που διοργάνωσε η Εθελοντική Ομάδα Δράσης ν. Πιερίας "Ο ΤΟΠΟΣ ΜΟΥ". Οι νικητές θα βραβευτούν σε ειδική τιμητική εκδήλωση, που θα πραγματοποιηθεί στον Καπνικό Σταθμό (Φλέμινγκ 8, Κατερίνη) το Σάββατο, 15 Μαρτίου 2014 και ώρα 19.00. Επαίνους για τη συμμετοχή τους θα λάβουν όλοι και όλες όσοι απέστειλαν διήγημα. Τα 23 παρακάτω διηγήματα που επιλέχθηκαν από την κριτική επιτροπή θα συμπεριληφθούν σε έναν συλλεκτικό τόμο, που θα απονεμηθεί στους νικητές και τις νικήτριες.
