# Εχουν χάσει τα αυγά και τα πασχάλια... (Γιώργος Τούλης)
Ηλία, άντε τυχεράκια, τζάμπα διαφήμιση σου κάνανε! Δεν μπορώ να πιστέψω ότι φτάσανε σε αυτό το σημείο. Βρε μήπως είστε συνεννοημένοι και η ιστορία είναι στημένη; Διαφορετικά δεν εξηγείται... Είναι να απορεί κανείς. Δεν βρέθηκε ένας δικός τους να τους σταματήσει και να τους πει ότι αυτό που σκέφτονται να κάνουνε είναι πατάτα και θα γυρίσει μπούμερανγκ εναντίον τους; Και δεν έφτανε μια μήνυση, έπρεπε να κάνουν έξι; Τι να πω, κρίμα... Αυτά δεν πρέπει να γίνονται. Το γεγονός όμως ότι γίνονται δείχνει ότι έχουν χάσει τα αυγά και τα πασχάλια... Συνιστώ στη Δημοτική μας Αρχή να διαβάσει το βιβλίο "Συναισθηματική Νοημοσύνη" για να μάθει να αντιμετωπίζει πιο ήπια τις κρίσεις ώστε να κερδίζει τουλάχιστον τις εντυπώσεις. Γιώργος Τούλης
