# Φέρτε μια κούπα …χωρίς κρασί!
Φίλες και φίλοι Στο προσωρινό κέντρο φιλοξενίας προσφύγων της Χράνης διαμένουν περίπου 300 άτομα, μεταξύ αυτών περισσότερα από 70 παιδιά, ηλικίας μερικών ημερών έως και 10 ετών. Στους ανθρώπους αυτούς προσφέρονται ημερησίως γάλα και τσάι, με αποτέλεσμα να καταναλώνονται περισσότερα από 1.000 πλαστικά ποτήρια μιας χρήσης, τα οποία καταλήγουν στα σκουπίδια. Σκεφτήκαμε να προσφέρουμε σε κάθε άτομο μια κούπα καφέ/τσαγιού την οποία θα επαναχρησιμοποιούν και έτσι θα μειωθεί σημαντικά η άσκοπη κατανάλωση πλαστικού που επιβαρύνει το περιβάλλον. Μάλιστα η εταιρία που έχει αναλάβει την σίτιση των προσφύγων είναι διατεθειμένη να διαθέσει μεγάλη ποσότητα αντιφθειρικών σαμπουάν ως ανταπόδοση για την εξοικονόμηση πόρων από την μείωση κατανάλωσης πλαστικών ποτηριών μιας χρήσης. Σας παρακαλούμε να συγκεντρώσετε τις κούπες καφές/τσαγιού που δεν χρειάζεστε και να τις φέρετε στον Καπνικό Σταθμό. Εκεί μπορείτε να τις αφήσετε στο πρώτο ξύλινο σπιτάκι της εισόδου. Σας ευχαριστούμε πολύ! Εθελοντική Ομάδα Δράσης Ν. Πιερίας
