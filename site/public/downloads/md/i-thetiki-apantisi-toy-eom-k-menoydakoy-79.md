# Η Θετική απάντηση του ΕΟΜ (κ. Μενουδάκου)
Αξιότιμε Κύριε Τσολακίδη, Αρχικά θα ήθελα, εκ μέρους του ΕΟΜ, να σας ευχαριστήσω και να σας συγχαρώ για τη πρωτοβουλία σας. Θα είμαστε στο πλευρό σας σε αυτή σας τη προσπάθεια. Ο ΕΟΜ μπορεί να σας διαθέσει ομιλητή για την Κυριακή 12/04/09 καθώς και να καλύψει τα έξοδά του. Σχετικά με τα βλαστοκύτταρα, ομιλητή για τα οποία θα υποδείξει το Νοσοκομείο της Κατερίνης, καλό θα ήταν να απευθυνθούν στο Ίδρυμα Ιατροβιολογικών ερευνών της Ακαδημίας Αθηνών και συγκεκριμένα στην κα Σταυροπούλου (τηλ. 210-6597252), είναι η ειδήμων πάνω στο θέμα και αν δεν μπορεί να έρθει η ίδια θα σας υποδείξει κάποιον. Σχετικά με το υλικό, μπορούμε να σας διαθέσουμε τα 5.000 και περισσότερα αν χρειάζεστε έντυπα (βλ. συνημμένο) και κάποια πανώ των οποίων τις διαστάσεις δεν γνωρίζω αυτή τη στιγμή. Οι αφίσες μας είναι μεγαλύτερες από Α2, είναι στο κλασσικό μέγεθος αφίσας και μπορούμε να σας διαθέσουμε 200 (βλ. συνημμένο). Παραμένω στη διάθεσή σας για τα περαιτέρω.
