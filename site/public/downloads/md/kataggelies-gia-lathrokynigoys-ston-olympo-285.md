# Καταγγελίες για λαθροκυνηγούς στον Όλυμπο
Μήπως θα μπορούσε να γίνει ένα κέντρο για καταγγελία κυνηγών που κυνηγάνε στον Όλυμπο;; ανεβαίνω αρκετά συχνά, όπως και πολύ κόσμος και όλοι οι ορειβάτες που δραστηριοποιούνται βλέπουμε αρκετά συχνά οχήματα κυνηγών και κυνηγούς. Είναι γελοίο να έχει στο δρόμο από Λιτόχωρο προς Πριόνια ένα κιόσκι και να ρωτάει μόνο την Άνοιξη ή το Φθινόπωρο που πηγαίνουν τα οχήματα. Είναι δύσκολο να βάλουν σε κάθε δρόμο προσέγγισης ένα άτομο να κάνει έλεγχο για κυνηγετικά όπλα και να απαγορευτούν τα σκυλιά στο Όλυμπο;; Από το δρόμο προς Χριστάκη γίνεται ουρά από κυνηγούς οι οποίοι όπως λένε πάνε να εκπαιδεύσουν τα σκυλιά τους!!!! ναι και εγώ είμαι ο superman. Επειδή ασχολείστε σοβαρά με αυτά τα θέματα θα παρακαλούσα να γίνει ένα newsletter από την οργάνωση σας και να προτρέπει όλο το ορειβατικό κοινό να στείλει φωτογραφικό υλικό με κυνηγούς και οχήματα και να φαίνονται και οι πινακίδες των οχημάτων να σταματήσει αυτή η κοροϊδία και όχι μόνο στον Όλυμπο αλλά σε κάθε Εθνικό Δρυμό. Αν θέλετε να βοηθήσω να βρείτε τις ηλεκτρονικές διευθύνσεις των ορειβατικών συλλογών και να τους αποστείλετε μια ομαδική επιστολή να έχουμε και υλικό στα χέρια μας όχι μόνο υπογραφές. Αν νομίζετε ότι μπορώ να φανώ χρήσιμο ενημερώστε με, ευχαριστώ πολύ για το χρόνο σας Π.Ν.
