# Μαίρη Κ.
Τέλος με βαθιά συγκίνηση θέλω να σας ευχαριστήσω όλους θερμά για όλα αυτά που έζησα και ένοιωσα μαζί σας και που μου δώσατε την ευκαιρία να βρεθώ ανάμεσα σε πραγματικούς ανθρώπους που δίνουν καθημερινά την ψυχή τους για να γίνει ο τόπος μας καλύτερος.Είμαι πάντα στη διάθεσή σας.Σας ευχαριστώ πολύ που είδα κοντά σας πως υπάρχουν άνθρωποι που δίνουν απλόχερα την αγάπη τους για ένα χαμόγελο! Θεσ/νίκη Μαίρη Κ.
