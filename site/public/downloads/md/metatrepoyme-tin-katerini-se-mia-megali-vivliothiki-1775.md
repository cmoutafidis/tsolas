# Μετατρέπουμε την Κατερίνη σε μια μεγάλη βιβλιοθήκη!
Η Εθελοντική Ομάδα Δράσης Ν. Πιερίας συμμετέχει στον εορτασμό της Παγκόσμιας Ημέρας Βιβλίου, στις 23 Απριλίου 2018 και φέρνει βιβλία σε κάθε γειτονιά της πόλης. Περισσότερα από 150 βιβλία κάθε είδους μπορούν να βρουν σε δημόσιους χώρους, οι μικροί και μεγάλοι αναγνώστες. Η Εθελοντική Ομάδα Δράσης Ν. Πιερίας μετατρέπει την Κατερίνη σε μια μεγάλη υπαίθρια βιβλιοθήκη. Βιβλία σε στάσεις λεωφορείου, βιβλία στις παιδικές χαρές, βιβλία στο δημοτικό πάρκο της πόλης μας, βιβλία στα ΚΤΕΛ, βιβλία στο σιδηροδρομικό σταθμό, βιβλία σε καφετέριες στον πεζόδρομο, βιβλία σε ξωκλήσια, βιβλία στις γειτονιές των Ευαγγελικών, βιβλία σε κούνιες, βιβλία παντού και βιβλία για όλους. Η Εθελοντική Ομάδα Δράσης Ν. Πιερίας, στο πλαίσιο της ανάπτυξης της φιλαναγνωσίας, άφησε βιβλία σε δημόσιους χώρους, τα οποία, αφού τα διαβάσετε, θα τα αφήσετε κι εσείς με τη σειρά σας με την ελπίδα να τα βρει ο επόμενος τυχερός αναγνώστης. Κάνουμε την Κατερίνη, μια μεγάλη βιβλιοθήκη. Αν λοιπόν βρεθείτε μπροστά σε ένα τέτοιο βιβλίο, μη διστάσετε..... Είναι δικό σας. Μπορείτε να το πάρετε μαζί σας. Καλή ανάγνωση
