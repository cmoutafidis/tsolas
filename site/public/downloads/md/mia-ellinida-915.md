# Μία Ελληνίδα
Βιομηχανικά απόβλητα στον Όλυμπο. Σας συγχαίρω! Έλληνες με καρδιά, για την απόφασή σας να ξεμπροστιάσετε έναν ανάξιο δήμαρχο και να αποτρέψετε την μόλυνση ενός τόπου ιερού για όλους εμάς τους Έλληνες. Είθε να αποτελέσετε φωτεινό παράδειγμα για ανάλογες περιπτώσεις όπου χρειαστεί, στη χώρα μας. Σας χαιρετώ και σας τιμώ. Μία Ελληνίδα
