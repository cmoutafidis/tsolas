# Μήπως θα μπορούσατε να βάλετε από ένα 100στάρικο;
Αξιότιμοι βουλευτές του Ν. Πιερίας Μήπως θα μπορούσατε να βάλετε από ένα 100στάρικο από τους παχυλούς σας μισθούς για να μεταφραστεί σε 5-6 γλώσσες μια σημαντική επιστολή , σχετικά με το τεράστιο χρέος της Γερμανίας απέναντι στη χώρα μας; Τα υπόλοιπα θα τα αναλάβουμε εμείς οι απλοί πολίτες! Θα το κάνουμε γνωστό σε ολόκληρο τον κόσμο, πως οι Γερμανοί θα έπρεπε να κατασκευάζουν τους ελληνικούς δρόμους αφιλοκερδώς και όχι να απαιτούν διόδια για δρόμους που πληρώθηκαν πολλάκις από τους έλληνες φορολογούμενους. Κύριοι βουλευτές, κάποιοι από εσάς οικονομολόγοι, ένας φυσιολογικός άνθρωπος, όταν ζορίζεται οικονομικά, το πρώτο πράγμα που κάνει, είναι να ζητήσει να του επιστρέψουν αυτά που του χρεωστούν. Εσείς γιατί δεν το πράττετε; Τα κόμματά σας γιατί σιωπούν; Γιατί δεν προτείνετε στον κ. Reichenbach να διαβάσει τον έγκριτο Spiegel και όσα υποστηρίζει ο ιστορικός οικονομολόγος Albrecht Ritschl για τα χρέη της Γερμανίας προς την Ελλάδα; Γιατί δεν χρησιμοποιείτε παραδείγματα εκατοντάδων γερμανών πολιτών οι οποίοι ντρέπονται για όσα επιβάλει η Γερμανία στη χώρα μας (βλέπε σχόλια γερμανών πολιτών στο ίδιο άρθρο); Πότε επιτέλους θα εκπροσωπήσετε τα πραγματικά συμφέροντα αυτής της χώρας και των πολιτών που σας επέλεξαν;
