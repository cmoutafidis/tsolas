# Σας ευχόμαστε ολόψυχα Καλές Γιορτές
Η Εθελοντική Ομάδα Δράσης Ν. Πιερίας, με τη δική σας βοήθεια και συμπαράσταση, θα συνεχίσει να αγωνίζεται για αξιοπρέπεια στη ζωή. Σας ευχαριστούμε που στηρίζετε τις κοινωνικές δομές του Καπνικού Σταθμού Κατερίνης. Σας ευχόμαστε ολόψυχα Καλές Γιορτές
