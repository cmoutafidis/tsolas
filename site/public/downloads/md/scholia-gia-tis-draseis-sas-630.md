# Σχόλια για τις δράσεις σας
της Δέσποινας Δραγωμανίδου (Χαλκιδική) Καλησπέρα / Καλημέρα σας. Πρόσφατα έτυχε να διαβάσω για κάποια δράση σας στον όμορφο νομό σας και σήμερα πάλι έπεσα πάνω σας, μέσω ενός μπλογκ. Είπα, λοιπόν, να κοιτάξω μήπως έχετε ιστοσελίδα και φυσικά σας βρήκα. Από τα λίγα που έχω δει μέχρι στιγμής, οφείλω να ομολογήσω ότι έχω μείνει άφωνη, αλλά και πολύ εντυπωσιασμένη από τις δράσεις και πρωτοβουλίες σας και γι’αυτό ένιωσα την ανάγκη να σας συγχαρώ όλους. Πραγματικά, σε τόσο δύσκολους καιρούς, ο εθελοντισμός είναι μια – ίσως η μόνη – ελπίδα για καλύτερες μέρες στη χώρα μας. Είμαι και γω της άποψης να μην τα περιμένουμε όλα από τους άλλους (τους έχοντες εξουσία), αλλά ότι μόνοι μας μπορούμε να φτιάξουμε τον κόσμο που ονειρευόμαστε. Σας εύχομαι να πρωτοπορείτε πάντα και να έχετε την ανταπόκριση του κόσμου – νομίζω πως κυρίως πρέπει να στοχεύσετε στους μαθητές, διότι είναι το μέλλον του τόπου. Και φυσικά εύχομαι να βρεθούν μιμητές σας σε κάθε νομό της Ελλάδας και να υπάρχει συναγωνισμός για τον πιο δραστήριο σύλλογο εθελοντών ! Ελπίζω και στα μέρη μου (ν. Χαλκιδικής) να ξεκινήσει κάτι αντίστοιχο, με φωτεινό παράδειγμα (και συμπαραστάτες) εσάς. Καλή εβδομάδα με υγεία πρώτα απ’όλα. Φιλικά, Δέσποινα Δραγωμανίδου (Χαλκιδικη)
